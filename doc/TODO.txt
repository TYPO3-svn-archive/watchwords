SHORT DESCRIPTION
=================
This extension adds a new content element which will get a new Christian Watchword every day.



LONG DESCRIPTIONS
=================
English
-------
This extension for the open-source content management system TYPO3 (www.typo3.com) provides a possibility to directly integrate watchwords into your website. All you need to do is to install the watchwords-extension on TYPO3. This will result in a new content-element that will fetch the current watchwords in an XML-format automatically from www.losung.de.

German
------
Für das Open-Source Content Management System TYPO3 (www.typo3.com) besteht die Möglichkeit, Losungen direkt zu integrieren. Dazu muss lediglich die Watchwords-Extension auf TYPO3 installiert werden. Die Losungen stehen dann als neues Content-Element zur Verfügung, welches die aktuellen Bibelverse jeweils automatisch im XML-Format von www.losung.de bezieht.
